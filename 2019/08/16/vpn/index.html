    Fake ip bằng Bitvise SSH + Proxifier

    PC windows
    Thời gian tạo:2019-08-16 23:48
    Số Từ:483 Số lần đọc:
    Nhận xét:

    Chắc hẳn sẽ có lúc các bạn cần phải thay đổi địa chỉ IP sang nước ngoài để làm công việc của mình. Bài viết này mình hướng dẫn fake IP bằng SSH với phần mềm Bitvise SSH Client và Proxifier. Mình xin nói qua về khái niệm SSH: SSH (tiếng anh là Secure Shell) là một giao thức mạng dùng để thiết lập kết nối mạng một cách bảo mật. 1 SSH thường có dạng như sau: IP (host)|user|password có thể có thêm vị trí, tên quốc gia của SSH đó tùy vào phần mềm Scan SSH xuất ra. Ví dụ mình có 1 SSH Nhật Bản có dạng : 61.123.148.179|test|test|Japan (JP)

    Bước 1: Tải:

    • Proxifier : https://www.proxifier.com/download.htm
    • Bitvise SSH : https://www.bitvise.com/download-area (chọn client)

    Bước 2: Cài proxifier + SSH (duh), crack proxifier cứ lùng trên mạng

    Bước 3: Bật proxifier lên:

    • Chọn Profile -> Proxy Servers -> Add -> Điền như cái hình
    • Tắt bảng Proxy Servers, chọn Profile -> Proxification Rules -> Add -> Điền tên & đường dẫn app bạn muốn fake ip vào như hình sau rồi bấm OK.

    Chú ý: Check trước khi add nếu fail xem lại tab Service trong SSH đã đúng dải ip 127.0.0.27 chưa nếu sai thì phải sửa lại mới chạy đc.

    Xong phần Proxifier, giờ qua SSH

    Bước 4: Vào trang SSH Monthly đăng kí tài khoản và chọn quốc gia muốn sử dụng (chọn free)

    Bật SSH lên:

    • Vùng server bên trái nhập địa chỉ host và port lấy từ SSH Monthly vừa tạo.
    • Vùng Authentication bên phải nhập vào username và password bạn vừa tạo.

    Điền như hình sau rồi bấm OK.

    Setup thành công -3- Sau này mỗi lần muốn chơi là bật proxifier + SSH login vào rồi vào app bạn cần, thấy bên proxifier có dòng app.exe là đang connect.

    Bạn cũng có thể save lại profile cho proxifier + SSH dành cho ai lười config lại
